# Student Manager 🎓

A desktop application built with Java Swing and SQLite for managing students and courses.

## Features

- **Student Management** — Add, update, delete, and view students with course assignments
- **Course Management** — Add, update, delete, and view courses
- **SQLite Database** — Lightweight, no server setup required
- **Tabbed Interface** — Clean navigation between Students and Courses tabs
- **Dashboard** — User selection panel with activity log

## Tech Stack

- Java 16
- Swing (GroupLayout, GridBagLayout)
- SQLite via sqlite-jdbc

## How to Run

### With Maven
```bash
mvn compile exec:java
```

### With NetBeans
Open the project, right-click `StudentManager.java` → Run File
